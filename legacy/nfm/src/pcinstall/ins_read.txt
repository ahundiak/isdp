Steps to generate the ninstall.exe and nremove.exe executable :
1. ftp >get ins_pc.tak and then take ins_pc.tak
2. nmk /f nmakefile
3. nmk /f rmakefile

Steps to generate the zip file cnfm.exe
1. may edit nfm.zip if the path for nfmpc.exe and nremove.exe is not
   the current directory
2. run makezip.bat to create cnfm.exe
   this will zip nfmpc.exe, README, nremove.exe

To run ninstall
 ninstall -p <absolute path> -h <nodename> [-d -o <debugfile>]
